## <a name="documentation"></a> Accès à la documentation

La documentation de SciPP est générée en utilisant [Doxygen](https://www.doxygen.org) à partir des commentaires du code source de SciPP et des divers fichiers Markdown disponibles sur le git (dont celui-ci).
Pour vous documenter sur SciPP vous pouvez donc:

* Lire les fichiers Markdown disponibles sur le git (la racine étant `README.md`)
* Lire la documentation générée avec Doxygen sur [mon site](https://www.reiex.site/SciPP)
* Lire la documentation après l'avoir généré soit-même avec Doxygen

Pour générer la documentation il suffit d'installer doxygen puis de lancer la commande suivante depuis le répertoire du projet:

```bash
make docs
```
