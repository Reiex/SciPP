# SciPP - Bibliothèque scientifique C++

## Description générale

SciPP est une petite bibliothèque scientifique développée en C++ prévue pour être utilisée avec la [SFML](https://www.sfml-dev.org/).

Elle implémente entre autres un module de gestion d'entiers (big int), un module de gestion de matrices et un module de dessin scientifique.
